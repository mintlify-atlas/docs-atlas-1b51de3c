The `runt` command-line tool provides utilities for managing notebooks, kernels, and the background daemon.

## Installation

The CLI is bundled with the nteract Desktop app. After installing nteract, the `runt` command is available in your PATH.

<Note>
On macOS, the CLI is located at `/Applications/nteract.app/Contents/MacOS/runt`. The installer automatically adds this to your PATH.
</Note>

## Quick Start

<CodeGroup>
```bash Open a notebook
runt notebook path/to/notebook.ipynb
```

```bash Launch interactive console
runt console
```

```bash List open notebooks
runt notebooks
```

```bash Check daemon status
runt daemon status
```
</CodeGroup>

## Command Structure

The CLI is organized into subcommands:

| Command | Description |
|---------|-------------|
| `runt notebook` | Open the notebook application |
| `runt console` | Launch an interactive Jupyter console |
| `runt notebooks` | List open notebooks with kernel info |
| `runt daemon` | Daemon service management |
| `runt jupyter` | Standalone kernel management |
| `runt shutdown` | Shutdown a notebook's kernel |

## Global Options

Most commands support:

- `--json` вҖ” Output results in JSON format
- `--help` вҖ” Show help for the command

## Usage Examples

### Open a notebook

```bash
# Open an existing notebook
runt notebook analysis.ipynb

# Open with specific runtime
runt notebook --runtime python new.ipynb
runt notebook --runtime deno script.ipynb
```

### Check daemon health

```bash
runt daemon status
runt daemon logs -f
```

## Exit Codes

Commands return standard exit codes:

- `0` вҖ” Success
- `1` вҖ” Error (daemon not running, kernel not found, etc.)
